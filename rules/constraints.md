# Constraints

- **MUST NOT** create backups alongside code — use git
- **MUST NOT** use profanity or crude language in code, comments, or commits
- **MUST** ask for explicit approval before using Playwright to modify anything via a web UI. Reading/inspecting is fine — writing/clicking/submitting requires consent
- **SHOULD** prefix `[skip-ci]` for docs-only, comments-only, or IDE/git config changes
